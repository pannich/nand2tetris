Project 9

=Instructions for Compilation:=
Use JackCompiler.sh <filedir>
- $ JackCompiler.sh Snake

=Instructions for Running the Code:=
- $ VMEmulator.sh

Functionality:
- Snake Game
- Snake eats randomly positioned apple.
- Snake grows as it eats apple.
- Snake dies when it hits the wall.
- Snake dies when it hits itself.
- Snake moves backwards will hit itself.

Configurations:
- Can adjust delay rate which is the speed of the game
- Can adjust 'togrow' rate in the Snake.jack

[List any known limitations or issues.]
- Memo.alloc stackoverflow after 3 minutes running
